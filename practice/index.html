List of GNU/Linux distributions recomend by me

Arch based

Manjaro Linux

Fast, user-friendly, desktop-oriented operating system based on Arch Linux. Key features - automatic hardware detection, stable rolling-release model, ability to install multiple kernels, special Bash scripts for managing graphics drivers and extensive desktop configurability.

Desktop: Awesome, bspwm, Budgie, Cinnamon, GNOME, i3, KDE Plasma, LXQt, MATE, Openbox, Xfce

Category: Desktop, Live Medium, Raspberry Pi

See on distrowatch

Official site
Artix Linux

Fork (or continuation as an autonomous project) of the Arch-OpenRC and Manjaro-OpenRC projects. Lightweight, rolling-release operating system featuring the OpenRC init software.

Desktop: Cinnamon, LXDE, LXQt, MATE, KDE Plasma, Xfce

Category: Desktop, Live Medium

See on distrowatch

Official site
Garuda Linux

Rolling distribution based on the Arch Linux. Performance-oriented, with many performance enhancing tweaks: zram, performance CPU governor, custom memory management software.

Desktop: bspwm, Cinnamon, GNOME, i3, KDE Plasma, LXQt, MATE, Recbox, UKUI, Wayfire, Xfce

Category: Desktop, Live Media

See on distrowatch

Official site

Debian based

MX Linux

Desktop-oriented Linux distribution based on Debian's "stable" branch, it is a mid-weight operating system with high stability, solid performance and medium-sized footprint.

Desktop: KDE Plasma, Xfce

Category: Desktop, From RAM, Live Medium, Raspberry Pi

See on distrowatch

Official site
antiX Linux

Fast, lightweight and easy-to-install Linux live CD distribution based on Debian's "Stable" branch for x86 compatible systems. Offers users the "antiX Magic" in an environment suitable for old computers.

Desktop: Enlightenment, GNOME, KDE Plasma, LXDE, MATE, Xfce

Category: From RAM, Live Medium, Old Computers

See on distrowatch

Official site
Parrot (formerly Parrot Security OS)

Security-oriented distribution featuring a collection of utilities designed for penetration testing, computer forensics, reverse engineering, hacking, privacy, anonymity and cryptography.

Desktop: KDE Plasma, MATE

Category: Forensics, Live Medium, Security

Official site

Independently distros

Arch Linux

Independently developed, x86_64-optimised Linux distribution targeted at competent Linux users. It uses 'pacman', its home-grown package manager, to provide updates to the latest software applications with full dependency tracking.

Desktop: Cinnamon, Enlightenment, GNOME, KDE, LXDE, MATE, Xfce

Category: Desktop, Server

Official site
Debian

Association of individuals who have made common cause to create a free operating system. Debian comes with over 50,000 packages - all of it free.

Desktop: AfterStep, Awesome, Blackbox, Cinnamon, Fluxbox, flwm, FVWM, GNOME, i3, IceWM, ion, JWM, KDE, LXDE, LXQt, MATE, Openbox, pekwm, Ratpoison, WMaker, XBMC, Xfce

Fedora

Gentoo Linux

Versatile and fast, completely free Linux distribution geared towards developers and network professionals. Unlike other distros, Gentoo Linux has an advanced package management system called Portage.

Desktop: Awesome, Cinnamon, Enlightenment, i3, GNOME, KDE Plasma, LXDE, LXQt, MATE, Openbox, Pantheon, Ratpoison, Xfce

Category: Desktop, Server, Source-based

See on distrowatch

Official site
NixOS

Independently GNU/Linux distribution that aims to improve the state of the art in system configuration management. In NixOS, the entire operating system, including the kernel, applications, system packages and configuration files, are built by the Nix package manager.

Desktop: Awesome, Enlightenment, Fluxbox, GNOME, i3, IceWM, KDE Plasma, Ratpoison, Xfce

Category: Desktop, Education, Live Medium, Server

See on distrowatch

Official site
Slackware Linux

advanced Linux operating system, designed with the twin goals of ease of use and stability as top priorities. Including the latest popular software while retaining a sense of tradition, providing simplicity and ease of use alongside flexibility and power, Slackware brings the best of all worlds to the table.

Desktop: Blackbox, Fluxbox, FVWM, KDE, WMaker, Xfce
