List of GNU/Linux distributions recomend by me

Manjaro Linux

Fast, user-friendly, desktop-oriented operating system based on Arch Linux. Key features - automatic hardware detection, stable rolling-release model, ability to install multiple kernels, special Bash scripts for managing graphics drivers and extensive desktop configurability.

Desktop: Awesome, bspwm, Budgie, Cinnamon, GNOME, i3, KDE Plasma, LXQt, MATE, Openbox, Xfce

Category: Desktop, Live Medium, Raspberry Pi

See on distrowatch

Official site
Artix Linux

Fork (or continuation as an autonomous project) of the Arch-OpenRC and Manjaro-OpenRC projects. Lightweight, rolling-release operating system featuring the OpenRC init software.

Desktop: Cinnamon, LXDE, LXQt, MATE, KDE Plasma, Xfce

Category: Desktop, Live Medium

See on distrowatch

Official site
Garuda Linux

Rolling distribution based on the Arch Linux. Performance-oriented, with many performance enhancing tweaks: zram, performance CPU governor, custom memory management software.

Desktop: bspwm, Cinnamon, GNOME, i3, KDE Plasma, LXQt, MATE, Recbox, UKUI, Wayfire, Xfce

Category: Desktop, Live Media

See on distrowatch

Official site
MX Linux

Desktop-oriented Linux distribution based on Debian's "stable" branch, it is a mid-weight operating system with high stability, solid performance and medium-sized footprint.

Desktop: KDE Plasma, Xfce

Category: Desktop, From RAM, Live Medium, Raspberry Pi

See on distrowatch

Official site
antiX Linux

Fast, lightweight and easy-to-install Linux live CD distribution based on Debian's "Stable" branch for x86 compatible systems. Offers users the "antiX Magic" in an environment suitable for old computers.

Desktop: Enlightenment, GNOME, KDE Plasma, LXDE, MATE, Xfce

Category: From RAM, Live Medium, Old Computers

See on distrowatch

Official site
Kali Linux

Kali Linux (formerly known as BackTrack) is a Debian-based distribution with a collection of security and forensics tools.

Desktop: Enlightenment, GNOME, KDE Plasma, LXDE, MATE, Xfce

Category: Data Rescue, Forensics, Live Medium, Raspberry Pi, Security

Official site
Arch Linux

Independently developed, x86_64-optimised Linux distribution targeted at competent Linux users. It uses 'pacman', its home-grown package manager, to provide updates to the latest software applications with full dependency tracking.

Desktop: Cinnamon, Enlightenment, GNOME, KDE, LXDE, MATE, Xfce

Category: Desktop, Server
